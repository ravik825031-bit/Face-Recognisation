# 🎯 Face Recognition System (Python + OpenCV)

A powerful and intelligent **Face Recognition System** built using Python, OpenCV, and Machine Learning.
This project detects, recognizes, and manages student data with an integrated GUI.

---

## 📌 Features

✨ Real-time Face Detection
🧠 Face Recognition using trained model
📸 Capture & store face samples
🗂️ Student data management (Add / Update / Delete)
📊 Attendance system with CSV export
🖥️ User-friendly GUI (Tkinter)
🔐 Secure login system (optional)

---

## 🛠️ Tech Stack

* Python 🐍
* OpenCV 👁️
* Tkinter 🎨
* NumPy
* PIL (Pillow)
* MySQL / SQLite (if used)

## ⚙️ Installation & Setup

### 1️⃣ Clone the repository

```bash
git clone https://github.com/your-username/face-recognition.git
cd face-recognition
```

### 2️⃣ Install dependencies

```bash
pip install opencv-python numpy pillow
```

### 3️⃣ Run the project

```bash
python main.py
```
